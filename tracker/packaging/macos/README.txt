ChipNomad - macOS Version
=========================

ChipNomad is a true chiptune tracker.

The build is not signed, so you will need to bypass macOS security checks:

1. Open Terminal and in the folder with ChipNomad.app run: xattr -dr com.apple.quarantine ChipNomad.app
2. Try opening ChipNomad, if macOS still doesn't open it, go to
   System Settings - Privacy & Security, and allow running ChipNomad.
